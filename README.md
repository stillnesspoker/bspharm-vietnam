# BSPHARM Vietnam Landing Page

Trang landing page tĩnh cho sản phẩm EVE EYES PRO+ theo phong cách product portfolio.

## Cách deploy miễn phí, tiết kiệm nhất

### 1) GitHub Pages
Phù hợp nếu bạn chỉ cần website tĩnh và đã có GitHub account.

- Chi phí: 0đ cho hosting
- Phù hợp: 1 landing page HTML/CSS/JS
- Hạn chế: cập nhật bằng cách push code lên GitHub

Cách làm nhanh:
1. Tạo repository mới trên GitHub.
2. Đưa toàn bộ file `index.html`, `styles.css`, `script.js` lên branch `main`.
3. Vào `Settings` -> `Pages`.
4. Chọn `Deploy from a branch`.
5. Chọn branch `main` và folder `/ (root)`.
6. Lưu lại, chờ GitHub tạo link public.

### 2) Cloudflare Pages
Thường là lựa chọn rẻ và nhanh hơn cho site tĩnh.

- Chi phí: 0đ cho hosting cơ bản
- CDN tốt, tốc độ ổn
- Có thể kết nối với GitHub để auto deploy

### 3) Netlify
Dễ dùng nếu cần preview nhanh và form đơn giản.

- Chi phí: 0đ cho mức cơ bản
- Rất nhanh để tạo link demo
- Hợp với landing page cần share ngay cho khách

## Khuyến nghị cho bạn

Nếu mục tiêu là tiết kiệm chi phí lâu dài, mình khuyên:

- Dùng **GitHub Pages** nếu chỉ cần website tĩnh rất đơn giản.
- Dùng **Cloudflare Pages** nếu muốn rẻ, ổn định và dễ mở rộng hơn.
- Dùng **Netlify** nếu cần tạo link demo cực nhanh để gửi khách ngay.
